## 終了

<img src="https://octodex.github.com/images/snowtocat_final.jpg" alt=celebrate width=300 align=right>

### おめでとうございます！このコースを修了しました！

リポジトリで完了したすべてのタスクを以下にまとめます。

- ベータリリースを作成する。
- リリースブランチに新しい機能を追加する。
- リリースプルリクエストを作成する。
- リリースノートを自動化する。
- リリースブランチをマージして確定する。
- リリースにホットフィックスをコミットする。
- リリース v1.0.1 を作成する。
